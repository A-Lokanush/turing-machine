# turing-machine
Repository of Formal Languages and Automata Theory Project

## Installing

```bash
git clone https://github.com/A-Lokanush/turing-machine.git
```

```bash
cd turing-machine
```

## Execute

```bash
g++ main.cpp -o main
```


## Run

```bash
./main input.txt
```

NOTE : Change your input in input.txt file
